# BRACU-Student-Routine-Management-System
Interactive Academic Calendar
 Display a clean, user-friendly calendar that organizes and shows all academic events—like quizzes, exams, presentations, and assignment deadlines—so students and faculty can easily track what’s coming up.

Comprehensive Event Tracking
 Let users keep track of all important academic events with relevant details and resource links. Faculty can add course-specific events tied directly to their courses, making it easy to stay aligned with the syllabus.

Grade Monitoring and Analysis
 Allow students to enter their marks for quizzes, assignments, and exams. The system will automatically calculate averages and monitor progress according to the grading scheme set by each course.

Smart Notification System
 Automatically send timely reminders about upcoming events. The system will also alert users if there’s a long gap without scheduled academic activities, helping everyone stay consistent and plan better.

Performance Visualization
 Show real-time visual breakdowns of student performance in each course with percentage scores and clear indicators based on the marks entered.

Centralized Course Materials Repository
 Enable faculty and students to upload, share, and access lecture slides, handouts, reference documents, and other materials linked to specific courses in one easy place.

Classroom and Social Features
 Faculty can create classrooms and add events, which automatically show up on the calendars of all enrolled students. They can also upload marks linked to those classrooms. Meanwhile, students can create social circles where they can view each other’s routines and stay connected.
